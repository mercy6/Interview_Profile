# 面試檔案集

## 專案介紹
這個資料夾是我為面試準備的專案文件，主要包括兩個部分：Python 程式開發和網頁設計。

## 資料夾結構

### Python
在Python資料夾中，包含以下兩個Python腳本：
- `crawler_pc_home.py`: 此腳本用於爬取PChome購物平台的暢銷排行和熱銷補貨信息。
- `crawler_weather_city.py`: 此腳本用於爬取台灣所有縣市的天氣數據。

### 網頁
網頁資料夾中包含三個子資料夾，分別是：
- `web01`: 由老師引導的網頁項目。
- `web03-1229`: 由老師引導的網頁項目。
- `期末作業`: 我個人獨立完成的網頁項目。
